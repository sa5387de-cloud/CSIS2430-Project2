# Project 2 Scope

## Project Goal

The goal of this project is to design and implement a Java program that demonstrates key programming concepts learned in CSIS 2430.  
The project focuses on building a clear program structure, implementing the required functionality, and verifying that the program runs correctly.

## Features

The program will include the following features:
- Read input data
- Process data using program logic
- Produce output results
- Handle basic edge cases

## Requirements

The program must:
- Compile successfully using Java
- Run from the command line
- Produce correct output for the required inputs
- Follow a clear and maintainable program structure

## Constraints

The program will follow the guidelines provided in the course assignment.  
Only standard Java libraries will be used.

## Out of Scope

The following features are not required for this project:
- Graphical user interfaces
- External frameworks or libraries
- Network communication

The focus of the project is correctness, clarity, and proper program structure.
