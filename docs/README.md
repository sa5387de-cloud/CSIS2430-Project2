# CSIS2430 Project 2

## Project Overview

This project implements and analyzes multiple sorting algorithms, including:

* Shaker Sort
* Quick Sort
* Merge Sort
* Heap Sort

The program generates permutations of input sizes and evaluates algorithm performance based on the number of comparisons.
